# O que significado CSS? #

    * Cascading Style Sheet
    * Codigo para criar estilos no HTML
    * HTML é a estrutura, e o CSS é a beleza
    * Não é uma linguagem de programação;
    * È uma linguagem style sheet

## Vamos ao exemplo

HTML -
<h1>Titulo Da pagina</h1>

```css 
    h1 {
        color:Blue;
    }
```
